# githoney-protocols

Tx3 models of [GitHoney](https://githoney.io/) protocols.

## Protocols

- [`githoney-bounties`](./githoney-bounties/) — fund, assign, merge, and claim on-chain bounties for open-source contributions.
- [`githoney-badges`](./githoney-badges/) — mint, update, transfer, and reclaim CIP-68 contributor and maintainer badges.
